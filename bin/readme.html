ImgDrive Portable ReadMe

File Description

FileDescription
imgdriveportable.exeA simple program that runs the imgdrive.exe in the following directory according to the current OS archtecture
imgdrive_x86\32-bit version of ImgDrive Portable
imgdrive_x64\64-bit version of ImgDrive Portable
imgdrive_arm64\ARM64 version of ImgDrive Portable

How to User

  1. Extract files
  2. Double click imgdriveportable.exe, or run imgdrive.exe in the corresponding architecture directly.
  3. To completely remove ImgDrive Portable you simply select "Exit and Remove Driver" in File menu.
